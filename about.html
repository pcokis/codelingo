About

  • Codelingo is an educational site with the focus of providing a platform for free and open source courses relating to computer science.

  • Each course contains a series of lessons with text and images providing examples, and each lesson contains at least one optional activity to follow.

  • This project is still in an early state, so content is still being added. Expect more courses and additional content to existing courses in the future!

  • All assets and the source code can be found on the github page.
