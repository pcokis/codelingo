What is Syntax?

Syntax is the way that code is written. While programming, you have to pay attention to spacing, capitalized characters, and symbols because if the code is not in the right format, a syntax error is returned.

Whitespace Matters

Whitespace includes spaces, tabs, and line breaks. In most programming languages, a semicolon is used to mark the end of a statement, but in python, you are able to use a line break to signify the end of a statement. Semicolons are still able to be used, but they are only helpful when using multiple statements under one line.
